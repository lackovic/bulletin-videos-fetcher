# Bulletin videos fetcher

Downloads all the videos from a video bulletin web page.

## Prerequisites

- [nvm](https://github.com/nvm-sh/nvm)

## Istall

1. Copy `.env.tpl` to `.env`

1. Edit `.env` and enter the bulletin URL

1. Run:

   ```sh
   nvm install
   nvm use
   npm install
   ```

## Run

```sh
npm start
```
